### Projets phares

#### [Anonymex](https://github.com/Reyder34/annonymex)
Application web d'anonymisation des copies d'examens, déployée et utilisée en production à la Faculté des Sciences de Montpellier. Cycle complet en équipe agile : conception, développement, déploiement, maintenance.
*Java · Spring Boot · REST API · MariaDB · Docker*

#### [ComptaExportNativ](https://github.com/Reyder34/ComptaExportNativ)
Outil C# / .NET livré à un cabinet de comptabilité : parsing, nettoyage et reformatage automatique de données tabulaires Excel vers le format d'import de l'ERP Divalto. Suppression complète des ressaisies manuelles.
*C# · .NET · OpenXML · Windows*

#### [Classification de tweets](https://github.com/Reyder34/Classification-tweet-scientifique)
Pipeline Python complet de classification supervisée : nettoyage, vectorisation TF-IDF, entraînement et comparaison de plusieurs modèles via scikit-learn (régression logistique, SVM, Naive Bayes). Évaluation rigoureuse par précision, rappel, F1 et matrice de confusion.
*Python · scikit-learn · Pandas · NumPy*

#### [Jeu de cartes multijoueur](https://github.com/Reyder34/ProjetReactApp)
Plateforme web temps réel : front React, communication bidirectionnelle Socket.IO, gestion des salles et synchronisation d'état côté serveur.
*React · JavaScript · Socket.IO · Node.js*

#### [LocusApp](https://github.com/Reyder34/LocusApp)
Application mobile Android développée en équipe.
*Kotlin · Android*
